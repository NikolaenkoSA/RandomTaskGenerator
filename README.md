## Описание программы
Random Task Generator — это GUI-приложение для случайного выбора задач из предустановленной или дополненной пользователем коллекции. Основные возможности:
- Генерация случайной задачи одной кнопкой.
- Категоризация задач по типам: учёба, спорт, работа, дом, хобби.
- Отображение полной истории сгенерированных задач с отметками времени.
- Фильтрация истории по типу задачи.
- Добавление новых задач в коллекцию с проверкой на пустой ввод.
- Просмотр полного списка задач в коллекции.
- Статистика использования (сколько задач сгенерировано, разбивка по типам).
- Очистка истории.
- Все данные сохраняются в JSON и загружаются при запуске.

Приложение работает на чистом Python без внешних зависимостей.
